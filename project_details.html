SignLingo - Real-time Sign Language Translation

Project Overview

SignLingo is a groundbreaking application that leverages the power of computer vision and machine learning to provide real-time translation of sign language into text and speech. Developed to bridge the communication gap for the deaf and hard-of-hearing community, this project showcases a sophisticated AI model trained on a vast dataset of sign language gestures. The user-friendly interface, built with React, ensures a seamless and intuitive experience for all users.

Key Features

  • Real-time gesture recognition and translation with high accuracy.
  • Text-to-speech output for audible communication.
  • Support for multiple sign language dialects through modular model architecture.
  • Customizable user interface with adjustable text size and themes.
  • Offline functionality for use in areas with limited internet connectivity.

Challenges & Solutions

Challenge: High-Variation in Gestures

Sign language gestures vary significantly between individuals and under different lighting conditions. This variability posed a major hurdle for achieving consistent model accuracy.

Solution: Data Augmentation & Normalization

We implemented an extensive data augmentation pipeline, creating synthetic variations of our dataset. Techniques like rotation, scaling, and brightness adjustments were applied. Additionally, hand landmarks were normalized to be scale and rotation invariant, drastically improving the model's robustness.

Technical Architecture

The system is built on a decoupled architecture. The frontend, a React application, captures video frames and sends them to a Python-based backend via WebSocket for low-latency communication. The backend processes the video stream using OpenCV, feeds it into a pre-trained TensorFlow/Keras model for inference, and sends the translated text back to the client.

Future Scope

Potential enhancements for SignLingo include expanding the vocabulary to cover more complex sentences, adding support for more sign language dialects globally, and developing a feature for two-way communication where spoken words are translated back into animated sign language. We are also exploring the possibility of deploying the model on edge devices for enhanced privacy and offline performance.
